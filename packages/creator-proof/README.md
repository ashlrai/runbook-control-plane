# `@runbook/creator-proof`

The fixed synthetic application profile for the first Creator Proof Sprint. It owns the rich seed charter, four allowed restrictive one-rule forks, deterministic policy evaluation, payload generation, and a separate domain receipt.

It never accepts brokerage data, tickers, returns, arbitrary files, arbitrary HTML, free-form claims, or a less restrictive fork. The parent capsule must still be verified by the caller; the profile receipt never changes the core Proof Capsule receipt or upgrades its assurances.
